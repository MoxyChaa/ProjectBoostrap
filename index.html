CSS

CSS, atau Cascading Style Sheets, adalah bahasa gaya yang digunakan untuk mengontrol presentasi atau tata letak halaman web yang ditulis dengan HTML dan XML. CSS memungkinkan pemisahan antara struktur dan desain suatu dokumen, memungkinkan pengembang web untuk membuat tata letak yang konsisten dan memudahkan pemeliharaan.

Tujuan utama Bootstrap adalah mempermudah pengembangan web dengan menyediakan kumpulan alat dan gaya desain yang konsisten.
